Library Management System
Overview
The Library Management System is a Java program designed to help libraries keep track of their members, books, and book issuances. It allows users to perform various tasks such as registering members, adding and removing books, and handling book issuances and returns.

Prerequisites
To run this program, you need to have the following installed on your system:
1. Java Development Kit (JDK)
2. Apache Maven (for building and managing dependencies)

Usage
Once the program is running, you can choose to enter as a librarian or a member.

Librarian
As a librarian, you have the following options:

1. Register a member: Add a new library member with their name, age, phone number, and issued books.
2. Remove a member: Remove a library member from the system.
3. Add a book: Add a new book to the library's collection, specifying the title, author, and number of copies.
4. Remove a book: Remove a book from the library's collection.
5. View all members: Display a list of all library members along with their books and fines to be paid.
6. View all books: Display a list of all books in the library.
7. Exit: Quit the program

Member
As a member, you have the following options:

1. List Available Books: Display a list of all available books in the library.
2. List My Books: Display a list of books you have borrowed.
3. Issue a Book: Borrow a book from the library's collection.
4. Return a Book: Return a book you have borrowed.
5. Pay Fine: View and pay any fines you may have accumulated.
6. Back: Go back to the main menu.

How to use : 
1. As a member or as a Librarian to select from menu, enter the index number.
2. To add a member or to add a book or to issue a book, enter the the required information about the book or member.
